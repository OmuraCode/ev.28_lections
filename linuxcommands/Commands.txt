Linux commands
Как открыть терминал - через spotlight - терминалы
~(tilda) - символ домашней директории

. - текущая директория

.. - предыдущая директория
/ - символ корневой директории
1. Ls- команда для вывода всех директорий(папкв) и файлов в текущей (rabochiy) direktorii v terminale

2. Cd (change directory) -  команда для перехода из одной директории в другую

3. Mkdir - создает директории 

4. Touch - создает файл

5. Rmdir - удаляет пустую директорию

6. Rm удаляет файл

7. Rm -rf удаляет вообще все

8. Ls -a показывает скрытые и обычные файлы и директории
   Ls -la показывает  подробную инфу

9. Pwd - выводит путь до текущей директории

10. Cp - копирует файл или директорию
   Cp <название> <куда>

11. Команда для перемещения файла или директории, а также для переименовывания файла или директории
        Переименовывание.  Mv John.txt jamie.txt
        Перемещение Mv John.txt jamie.txt/

12. Cat - показывает содержимое файла
    Копирование содержания файла - cat test.txt > test1.txt

13. Nano открывает файл для редактирование в граф интерфейсе, нано нет файла то нано создаст новый 

14. Echo -  команда для быстрой записи текста в файл
    Echo текст > файл

15. mkdir my_dir && cd my_dir - && следующая команда

16. useradd - добавление нового пользователя

17. Ctrl + C - сочетание клавиш используется для остановки определенного процесса

18. Ps - команда используется для вывода запущенных процессов

19. !! - символ для последней введенной команды,

20. Ls -l teest.txt - выведите в терминал объем занимаемой памяти файла test.txt

21. Env - команда для выведения списка переменных
